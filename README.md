Nostra — Fashion E-Commerce Website
Created by Balakrishnan

About
Nostra is a fashion e-commerce website built with HTML, CSS, and JavaScript.
It includes a home page, collections page, and contact page — designed for both mobile and desktop.

Pages

index.html — Home page with hero slider, brands, new arrivals, most wanted, and offers
collection.html — Shop page with category filters and search
contect.html — Contact form page


Features

Hero image slider with auto-play
Mobile sidebar menu (slides in from the right)
Like / favourite button on Most Wanted products
Category filter — Shirt, Half Shirt, Pants, Half Pants, Coat
New & Old arrival filter
Live search bar in collections
Fully responsive — works on mobile and laptop
